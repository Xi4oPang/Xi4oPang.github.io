准备

0x1.登录cloudflare 平台
0x2.找到联网—> Tunnels
0x3.创建隧道—> 下载 —> 安装 在关闭创建面板（主要是为了下载cloudflared-windows-amd64.msi）
0x4.双击运行创建得文件
0x5.我们打开powershell 验证一下

本地配置

0x1. 登录

cloudflared tunnel login 登录，这一步需要对你的域名进行授权

回到PowerShell窗口看到如下输出我这一步就成功了,然后会得到一个文件夹C:\Users\<你的用户名>\.cloudflared 。打开文件夹能看一个cert.pem的文件。

0x2. 创建隧道

使用如下命令创建隧道，隧道名自己随便起

终端输出如下：


这里的 ID 和 JSON 文件都会在下已被被使用。

0x3.配置隧道

再去Tunnels 面板检查一下是否有个同名的隧道被创建

0x4.运行

根据自己的情况修改path，等Console输出完后去Tunnels 面板看刚才命名的隧道 Status 变为 健康

0x5.添加DNS记录

Type: CNAME
Name: <你的隧道名称>
Target: <Tunnel ID>.cfargotunnel.com
填写好后保存即可

0x6.被远程电脑设置

1.“设置” —>“系统” —> “远程桌面” 点击启用远程桌面，再点击 “高级设置” 取消勾选 “需要计算机使用网络级别身份验证进行连接（建议）”
2.”账户” —> “登录选项” —> “密码” 创建一个登录密码

[!warning] 注意
如上操作都是被控制电脑所需要设置的

0x7.远程

1.远程电脑也需要安装 ‘cloudflare.exe’
2.运行隧道桥梁命令

3.Win + R 输入 mstsc 打开远程,在 计算机 填入： localhost:<Port> 点击连接，在填写用户名和密码即可，在”图二“中点击”是”就可以愉快的操作了。
