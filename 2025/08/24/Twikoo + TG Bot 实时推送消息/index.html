1. Telegram Bot

1.1 创建 Bot

  1. 在 Telegram 搜索 @BotFather。
  2. 输入 /newbot → 设置 Bot 名称和用户名。
  3. 获取 Bot Token（例如 123456789:ABC-xyz）。

1.4 实时推送方式

2.3 Twikoo 配置示例（带 Telegram 推送）

在 Twikoo 后台数据库配置 中添加：

  • MAIL_ENABLED: false 避免邮件模块报错。
  • 只启用 Telegram，评论消息会实时推送到你的 Chat。

2.4 常见问题

  • 评论按钮灰色不可点

    • 前端没正确配置 envId 或 api。
    • 检查浏览器 Console 是否报错。

  • AM Twikoo: 邮件初始化异常：数据库配置不存在

    • 邮件模块未配置，但被启用。
    • 解决：设置 MAIL_ENABLED: false 或补齐邮件配置。

  • Telegram 推送失败

    • 核心是 TELEGRAM_BOT_TOKEN 和 TELEGRAM_CHAT_ID 配置正确。
    • 若服务器在国内，可能需要代理。

总结

  • Telegram 推送消息的核心：Bot Token + Chat ID + sendMessage API。
  • Twikoo 评论不可点击按钮 → 前端配置错误，需确认 envId 或 api。
  • 推送失败常见于 服务器访问不了 Telegram API，可用代理解决。
