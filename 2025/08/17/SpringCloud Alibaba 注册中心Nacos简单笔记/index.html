搭建Nacos

1.安装Nacos(2.3.2)

1.1通过官网下载或者GitHub

我这儿下载的2.3.2版本，根据你需要版本下载即可。下载完毕后会得到一个压缩包nacos-server-2.3.2.zip 将其解压到指定目录即可

1.2配置和Nacos 配置文件

1.2.1修改startup.cmd 文件

set "JAVA=%JAVA_HOME%\bin\java.exe" 修改Java对应版本，如果使用Nacos 3.x以上的版本需要java17，startup.cmd中默认使用集群模式将 set MODE="cluster" 修改为单机模式 set MODE="standalone" 也可以通过cmd命令：startup.cmd -m standalone 启动为单机模式
